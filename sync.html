Cross-Device Sync (optional)

Sync your favorites, personal macros, and quiz progress across phone and workstation with your own free Firebase project. No patient data is ever stored — only your stars, templates, and study progress.

🔒PrivacyWhat syncs

Only favorites, macros, quiz boxes, and recently-viewed slugs — all non-clinical. Your AI API keys and any report text are never synced.

1 · Create a Firebase project (5 min, one time)

  1. Go to console.firebase.google.com → Add project.
  2. Build → Authentication → Sign-in method → Google → Enable.
  3. Build → Firestore Database → Create database (production mode).
  4. In Firestore → Rules, paste the rules from FIREBASE_SETUP.md (each user can read/write only their own doc) and Publish.
  5. Project settings (gear) → Your apps → Web (</>) → register an app → copy the firebaseConfig object.
  6. Add your GitHub Pages domain under Authentication → Settings → Authorized domains (e.g. stephen-12345.github.io).

2 · Paste your config

Paste the firebaseConfig JSON (the part inside the braces, or the whole { ... }). It is stored only on this device.
